URL endpoints and request methods

Request Endpoint URL HTTP Method Query parameters Request body data format Response body data format
Return list of all movies /movies GET None None JSON object holding data about all movies
Example: [{"title": "Your Name", "year": 2016, "genre": "Drama", "director": "Makoto Shinkai"}, {...}, ...]
Return list of movies by a certain director /movies/directors/[name] GET :name None JSON object holding data movies from a specific director
Example: [{"title": "Your Name", "genre": "Drama", "director": "Makoto Shinkai"}, {"Title": "Weathering with You".......}, ...]
Return list of movies by a certain genre /movies/genres/[name] GET :name None JSON object holding data movies from a specific genre
Example: [{"title": "Your Name", "genre": "Drama", "director": "Makoto Shinkai"}, {"Title": "Weathering with You".......}, ...]
Return info about specific movie by title /movies/[title] GET :title None JSON object holding data about requested movie
Example: {"title": "Your Name", "year": 2016, "genre": "Drama", "director": "Makoto Shinkai"}
Return info about a genre /genres/[genre] GET :name None JSON object holding data about requested genre
Example: Relationships and character development
Return info about a director by name /directors/[name] GET :name None JSON object holding data about requested director
Example: name: 'Makoto Shinkai', bio: 'Makoto Shinkai was born on February 9, 1973. He is a Japanese animator, fimlmaker, author, and manga artist.',
Add new user /users POST :users JSON object holding user data
Example: { "name": "Glacies", "favoriteMovies": ["Your Name", "A Silent Voice"] } 		JSON object holding the new user's data(Username, password, and email are required)
Example: { "Username": "Glacies", "Password":"password123","Email":"glacies@gmail.com", "Birthday": "1991-02-11", "favoriteMovies": [ "Your Name", "A Silent Voice" ]}
Update user information /users/[Username] PUT :id JSON object holding new user's info
Example: { Username: String, Password: String, Email: String, Birthday: Date } 		JSON object holding the new user's data
Example: { id: 1, name: 'Astrid', favoriteMovies: ['A Silent Voice'], }
Add a movie to a user's list of favorites /users/[Username]/movies/[movieID] Post :Username, :MovieID None text output confirming requested action
Example: "Summer Wars has been added to user 2's array."
Remove a movie from a user's list of favorites /users/[Username]/movies/[MovieID] DELETE :Username, :MovieID None text output confirming requested action
Example: "Wolf Children has been removed from user 4's array."
Delete user /users/[Username] DELETE :Username None text output confirming requested action
Example: "User 1 has been deleted."
